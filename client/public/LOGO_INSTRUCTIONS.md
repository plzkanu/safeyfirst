# SOOSAN 로고 이미지 추가 방법

## 이미지 파일 위치

SOOSAN 로고 이미지를 다음 위치 중 하나에 추가하세요:

- `client/public/logo.png` (권장)
- `client/public/logo.jpg`
- `client/public/logo.svg`
- `client/public/soosan-logo.png`
- `client/public/soosan-logo.jpg`
- `client/public/soosan-logo.svg`

## 지원 형식

- PNG (권장)
- JPG/JPEG
- SVG

## 권장 사양

- **크기**: 높이 40-60px (비율 유지)
- **배경**: 투명 배경 (PNG/SVG 권장)
- **해상도**: 고해상도 (Retina 디스플레이 대응)

## 이미지 추가 후

이미지를 `client/public/` 폴더에 추가하면 자동으로 로고가 표시됩니다.

이미지가 없거나 로드에 실패하면 텍스트 로고로 자동 대체됩니다.
